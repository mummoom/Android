## 기획 배경
- ***강아지에게 위험한 식재료인줄 모르고 먹였다가*** 병원에 급하게 데려가는 슬픈 일들을 인터넷에서 많이 보았습니다.
- 간식을 만들어주거나 또는 야채나 과일 등을 주고 싶은데 ***강아지가 어떤 식재료를 먹어도 되는건지*** 매번 검색하는게 번거로웠습니다.🤔
- 대형 반려견 카페 등에서 직접 만든 간식을 자랑하고 싶어하는 사람들이 정말 많았습니다!<br><br>
